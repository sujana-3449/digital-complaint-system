# 📌 Digital Complaint Handling and Tracking System

## 📖 Project Overview
The Digital Complaint Handling and Tracking System is a web-based application developed using Flask that allows users to register complaints, track their status, and receive updates efficiently.

## 🎯 Objectives
- Digitize complaint registration
- Provide real-time tracking
- Improve transparency
- Reduce manual work

## 🛠️ Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite
- Bootstrap

## 🚀 Features
- User Registration & Login
- Submit Complaints
- Track Status
- Admin Dashboard
- Status Updates

## 📂 Project Structure
- app.py
- templates/
- static/
- database.db

## ⚙️ How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/
